Este projeto foi desenvolvido como parte de um teste técnico.

COMO RODAR O PROJETO LOCALMENTE

1) Instalar as dependências

Na raiz do projeto, execute o comando abaixo para instalar todas as dependências necessárias:

npm install


2) Rodar o projeto

Após a instalação das dependências, execute:

npx vite


ESTRUTURA IMPORTANTE DO PROJETO

- Pasta "questionário"
  Nesta pasta estão as respostas do teste técnico, conforme solicitado no desafio.


OBSERVAÇÕES

- É necessário ter o Node.js instalado (recomendado versão 18 ou superior).
- Caso ocorra algum erro ao iniciar o projeto, tente:
  - Apagar a pasta node_modules
  - Rodar novamente: npm install
